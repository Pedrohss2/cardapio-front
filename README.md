## Como Rodar o Projeto

Siga os passos abaixo para rodar o projeto localmente:

### 1. Instalar dependências
No diretório do projeto, execute:

```bash
npm install
# ou
yarn
# ou
pnpm install
```
Rodar o projeto
```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
```
O projeto estará disponível em http://localhost:3000
